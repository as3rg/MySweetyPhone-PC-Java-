Меню слева содержит следующие разделы: “Избранное”, “Устройства”, “Подключиться”, “Новая  Сессия”, “Помощь” и “Выход”.
 - Раздел “Избранное” нужен для отправки текстовых сообщений и файлов на сервер для доступа к ним со всех устройств.
- Раздел “Устройства” позволяет просматривать и удалять устройства, привязанные к данному аккаунту.
- Раздел “Подключиться” позволяет данному устройству подключаться к другому.
- Раздел “Новая Сессия” позволяет создавать сессии для управления этим устройством.
- Кнопка “Выход” завершает работу приложения и выходит из аккаунта. При следующем запуске потребуется повторная аутентификация.

Избранное

Для отправки текстовых сообщений и файлов на сервер для доступа к ним со всех устройств используется вкладка “Избранное”. В ней можно отправлять сообщения, файлы, весом не более 1 Мб,  а также скачивать эти файлы на любом устройстве (пункт в контекстном меню).

Offline режим

Offline режим используется при отсутствии интернета. В данном режиме отсутствует вкладка”Диспетчер устройств” и вкладка “Избранное”. Чтобы перейти в offline режим, нужно при запуске приложения нажать на кнопку “Включить offline режим”.

Сессии

В инструкции ниже управляемое устройство (открывающее сессию), называется сервером, а управляющее устройство (подключающееся), называется клиентом.

Для работы следующих функций нужно быть подключенным к одному  Wi-Fi на клиенте и сервере.
В случае некорректной работы функций попробуйте отключить стандартный брандмауэр в Windows и/или антивирус. Возможно, они блокируют попытки соединения устройств.


Сессии могут выполнять разные функции: “Эмуляция ввода”, “Удалённый просмотр файлов”, “Удалённая отправка SMS”,“Отправка в буфер обмена”, “Удалённое открытие ссылки”.

Эмуляция ввода

Откройте сессию на сервере, выбрав “Эмуляция ввода” в выпадающем списке, подключитесь к данной сессии на клиенте.
В появившемся окне в правом верхнем углу будет предоставлен выбор функции: “Мышь”,”Клавиатура” или  “Графический планшет”.
- При выборе функции “Мышь” клиент может управлять курсором сервера (короткое нажатие – ЛКМ, длинное нажатие – ПКМ, свайп двумя пальцами – колёсико мыши, клик + свайп – перемещение мыши с зажатой ЛКМ(drag)).
Сервером для данной функции может быть только компьютер, а клиентом как компьютер, так и Android устройство.
- При выборе функции “Клавиатура” клавиатура клиента может заменять клавиатуру сервера. Сочетание клавиш Ctrl+Print Screen сохраняет скриншот с сервера на клиенте.
Сервером для данной функции может быть любое устройство, клиентом также любое устройство.


- При выборе функции “Графический планшет” Android устройство может использоваться в качестве графического планшета. Данная функция не требует настройки в графических редакторах и лучше всего работает в горизонтальном положении устройства-клиента.
Также можно выбрать область для рисования, для этого нажмите правой кнопкой мыши по иконке в трее и выберите соответствующий пункт.

Просмотр файлов

Откройте сессию на сервере, выбрав “Просмотр файлов” в выпадающем списке, подключитесь к данной сессии на клиенте.
Клиент получает доступ к просмотру, скачиванию и удалению файлов сервера, созданию новых папок и отправке файлов с клиента на сервер.

Просмотр SMS

Откройте сессию на сервере, выбрав “Просмотр SMS” в выпадающем списке, подключитесь к данной сессии на клиенте.
Данная функция позволяет удалённо читать и отправлять SMS.
Сервером для данной функции может быть только Android устройство, а клиентом как компьютер, так и Android устройство.

Открытие ссылки на компьютере

Для открытия на компьютере выделенной на Android устройстве ссылки нажмите кнопку “Share”, выберите приложение MySweetyPhone. Далее предоставляется выбор: отправить её во вкладку “Избранное”, открыть данную ссылку в браузере компьютера или отправить её в буфер обмена компьютера. Выберите пункт “Открыть на компьютере”, нажмите “Выполнить” и выберите компьютер.

Отправка текста в буфер обмена

Для отправки текста в буфер обмена компьютера, скопируйте нужный текст в Android устройстве, нажмите на кнопку “Share”, выберите приложение MySweetyPhone. Далее можно отправить данный текст во вкладку “Избранное” или в буфер обмена компьютера. Выберите пункт “Отправить в буфер обмена компьютера”, нажмите “Выполнить” и выберите компьютер.
